# hx-math
Vector, Matrix, and Math library for 4D, 3D, and 2D Computer Graphics

## Data types
### Vector
- vec5f, vec5i
- vec4f, vec4i, vec4u
- vec3f, vec3u
- vec2f, vec2i, vec2u

### Matrix
- mat5
- mat4
- mat3

### Axis-Aligned Bounding Box
- aabb (float)
- aabbi
